System Configuration

All the settings on this page are related to the pedestal not the reflector.

Autosat load on power up and search failure use to be part of the DAC system type. Which was system type 1 and 4.

Advanced settings are greyed out and are not editable fields. These settings are set via the system profile or via CLI commands.

Please hover over the different names for more info.

Miscellaneous

Sat Ref Mode  Sat Ref default is off, this function should be used when there is no gyro connected or there is an issue with the gyro for example:
slow updating nmea gyro, lagging gyro ( delay between vessel turning and the gyro signal changing), gyro read errors.

 		On   Off
 
Auto Sat Load Both of these should on
Power Up means when the antenna is switched on it will retarget the last used satellite.

Search Failure means when the antenna has finished a search pattern the antenna will retarget the satellite and start trying to find the satellite again.

Power Up  On   Off
Search Failure  On   Off
Profile Number:  Profile Number is what the N0 PCU parameter used to be. 1 = 4012-91. This is linked to the profile page.

Version is the version of the profile list, 14 means the list has been changed 14 times, this may be due to changes to the antenna or just adding new antennas to the profile list
