Logging In

Connect a PC to one of the lan ports on the MXP, set the PC IP address to 10.1.1.101. Then web browse to 10.1.1.100 which is the default MXP IP address.

To log in to the system use one of these three. Beware these are case sensitive.
  • Dealer with password seatel3
  • SysAdmin with password seatel2
  • User with password seatel1
For this example we will log in as "Dealer" and the password "seatel3".

It is recommended by Cobham Satcom that the passwords should always be changed from the default ones when the system is commissioned.
