Three Access levels

There are three categories of accessible menus.
  • Dealer
  • SysAdmin
  • User

The Dealer:

  • Has the most accessible selections.
  • Change the SysAdmin and User passwords.
  • Setup Wizard.

The Dealer or SysAdmin can log in while the User is using the system.
When the Dealer or SysAdmin logs out, all logged in users are logged out at the same time.
