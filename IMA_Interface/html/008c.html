Interface Configuration

This section allows you to change anything which is related to the serial ports.
These ports are a mixture of 9 way D types, RJ45 sockets and screw terminals.

Compared to the DAC's, there is more control and a larger selection of baud rates.

These ports are used for serial comunication to devices on the antenna, typicaly these would be BUC's. When the baud rate is change it will be change at the MXP and ICU. Click here for the image.
These are located on the rear of the MXP.
  • Aux 232
  • Console is the new M&C from the DAC's, the protcal is selected under Console which has been highlighted in red.
  • NMEA is for heading input using NMEA0183 standard
  • OBM/Modem is used for Modem and Out of Band Managment in the same way as it was on the terminal mounting strip
This port is located on the ICU.
Console is the new M&C from the DAC's, the protcal is selected under Console which has been highlighted in red.
Click here for the image.
This is where the Console protacol is changed, if CLI is selected then only CLI commands can be used on the Console ports. These drop down lists allows the option to output different types of NMEA sentances. GPS are a must becuse they are needed by the Vsat modem.
If any of the others are seleceted then that Nmea sentance must be coming in to the MXP before it can be retansmitted out, the GPS comes from the internal GPS fitted to the antenna.
The seconds let you select how oftern the sentence is to be transmitted. This is where the USB protacol is changed, if CLI is selected then only CLI commands can be used on the mini USB port. Select to the correct gyro type for the gryo which is connected to the MXP.
1:1 and NMEA are the only ones you do not need to enter a start heading.
NMEA you will need to select what type of NMEA string is being used from the dropdown list and which port is being used. Default is internal, the system will use the GPS on the antenna.
If you wish to use an external GPS then select which port the GPS is connected to and the GPS string being used.
Auto will automaticly select which has GPS on it. This is where you can enter or change the GPS to see if it's updating.
Please hover over the different names for more info and change the settings via the dropdown lists.
