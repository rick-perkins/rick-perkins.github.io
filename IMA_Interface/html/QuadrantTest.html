Four Quadrant Test

Manual  Auto
                     
  
This test allows the user to check the tracking performance by moving the antenna slightly off satellite and then allowing tracking to peak the antenna back on satellite. This test is affected by Dishscan and motor gains.
  • Turn tracking off.
  • Drive the antenna in one direction only so the AGC drops by 100 counts.
  • Turn tracking back on and the antenna should peak back up on satellite with in 30 seconds.
  • Repeat the test so EL and AZ has been tested in a up and down direction.
