Interface Configuration

This section allows you to change anything which is related to the IP network.

The MAC Address is the only field that can not be changed

Once the IP address, Subnet, Gatway or DNS has changed and the save button has been pressed, the changed field will become RED. At this point the settings will not take affect untill a MXP REBOOT

  Port These are the default port numbers. The port numbers are linked to the mode and port 2003 which CLI is a new port, the rest are the same as on the DAC2202 and DAC2302.   Mode As default port 2000 and 2001 are Legacy, 2002 is OpenAMIP and the new for the IMA electronics is 2003 which is CLI Secure  The padlock icon is used to show if the port is locked or unlocked. If the port is locked then the user name and password wil need to be entered to activate the connection.
