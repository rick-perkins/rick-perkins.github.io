These settings have the same functions as the settings on Satellite page under Configuration. Longitude is the satellite GEO location in East or West.
Frequency is the tracking frequency in MHz.
Skew is if the targeted/ tracking satellite has a known skew value.
Search pattern is for the targeted/ tracking satellite.
  • Inclined is for a satellite in an inclined orbit
  • Sky search is for no gyro mode
  • Sprial is for normal operation
 These settings have the same functions as the settings on Satellite page under Configuration.
TX Polarity changes the transmite polarity
Band changes the RF range of the LNB
Reflector is used for the Dual Band systems
RX switches between Co and X pol LNB's. The Threshold works in the same way it did for the DAC's.
Auto mode allows the user to turn on or off this function.
Auto offset is calculated in the same way as before, typicly this value will be 1/3 of differance between on and off satellite.
Threshold is calucalted everytime the satellite is targeted unless this functoin is turned off.
When the Calculate button is pressed on the reflector page the antenna will retarget and calculate the offset and threshold value automaticaly. These inputs allow the user to move the antenna by the number entered for each axis. Polang Target will be either a positive or negative value and we use 90 instead of 120 or 130 on the legacy systems.
Drive will change to Manual if the setting has been changed to manual on the Reflector page under configuration.
Linear and Circular will dispaly the pol angle, when in manual mode these will not be greyd out and to drive the pol all the user needs to do is enter in the pol angle required.
