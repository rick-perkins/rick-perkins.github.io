System Status

This page is a overview of the tracking satellite settings, system status and where errors can be viewed.

If an error is active, the word ERRORS will be red. If you click on it, a pop up window will appear displaying all the active errors.
All the errors are decoded unlike the older systems where you had to decode them.

Modem RX Lock will have a green indicator and display LOCKED when the Vsat modem has RX lock, if this is reversed change the settings on the interface page.

TX Mute will have a green indicator and display OFF when the MXP is not muting the Vsat modem TX, a red indicator will display TX mute as ON.

Front Panel LED is a mirror image of the LED indicators on the front panel of the MXP.

The GPS will update as the vessel changes location, the GPS source will depend on what has been selected on the interface page.

Cross level displays the cross level angle, this should always be 0.0 degs becasue this means the cross level is level with the horizion.

Polang is displayed in + and - unlike the old systems which always had a positive reading.
