Display Overview

The dial shows the orientation of the vessel, in relation to True North.

In this example the vessel is heading toward True North.

The red or white (when its not on the same value as the heading)line is the antennas azimuth value from True North.

  • Satellite Longitudinal position
  • Heading of the vessel
  • Azimuth of the antenna
  • Elevation angle of the antenna
  • Relative position of the antenna
  • Linear polang value of the polarization
The Status line will show one of six conditions:
  • Tracking Green LED
  • Tracking Orange LED for AGC below Threshold
  • Tracking Off LED off
  • Targeting Green LED
  • Searching Green LED
  • Disconnected Red LED
  • Initializing Green LED
The second line will show one of six conditions:
  • Normal Green LED
  • Appr. Blockage Flashing Orange LED
  • Blockage Flashing Red LED
  • TX mute Red LED
  • Stability Limit Red LED
  • Slow Scan
The Modem line will show one of two conditions:
  • Locked Green LED
  • Unlocked Red LED
