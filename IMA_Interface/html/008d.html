Interface Configuration

Modem Lock

This section allows you to set up the MXP to interface with the installed Vsat modem. On the old systems, this was setup by the DAC system type (System type 2,16 and 128).

The common Vsat modems are easily setup by selecting the installed modem from the modem type list, followed by how the modem is connected to the MXP via the Modem I/O list.

The modem I/O list will change depending on which modem type has been selected.

  • When Modem RJ45 is selected, the Voltage and 12V pull up's will be DISABLED.
  • The Voltage is used to adjust the lock sensing voltage for the Lock input signal.
  • If pull up voltages are needed, then TS1 or TS2 should be used and connections should be via J12.
  • If the modem is using openAMIP make sure this is set to openAMIP otherwise the MXP will not use the openAMIP commands.

OpenAMIP and ROAM is via the ethernet connection, even CLI control can be done this way.

The Reflector is not selectable via this menu, this is changed via the Position Antenna page.

  • If this is a singal band system (C or Ku only) then only Primary will be used and Secondary will not be accessable.
  • If this is a 9711IMA system with C and Ku band, both Primary and Secondary will need to be configured for the modem/s which are connected.
  • When using inconjuction with a SeaTel arbitrator make sure the Block ouput is always set to Polarity High.
