AZ Test

Zoom Level   1    2   3   4                Slow Scan Vlimit  deg/sec         
This test is used to check the AZ bearings and friction.
When the test is started, the antenna will drive slowly in AZ for a full 360°, then stop and drive back in the oppersite direction.
To stop the test use the Stop Test button, if this is not done the system will stay in Slow scan mode.
