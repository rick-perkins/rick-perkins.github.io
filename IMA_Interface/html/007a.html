Satellite Configuration

  • To edit a satellite use the drop down list select satellite.
    Then the satellite information will be displayed for editing.
  • To add a new satellite click on the Add Satellite button then enter in the correct information.
  • To save the satellite click on the Save button.
  • To delete a satellite, select it via the drop down list then click on Remove.
Hover over the input fields for more information.

 

Satellite Selection

  Select a satellite from the list to edit or delete.
Once selected all the satellite settings will be displayed below ready for editing or deleting  
If you do not see a dropdown list or your desired satellite is not in the dropdown list, click   
to add a satellite

Add Satellite

Satellite Name  Enter the name you wish the satellite to be saved as.
Longitude  Enter Satellite longitude with decimal place if needed followed
by E = East or W =West 		 
Freqency  Enter Satellite tracking freq in MHz · KHz  MHz
Skew  Enter Satellite skew value if known.
This is for satellites in an inclind orbit 		 deg
 
 
 
Search Pattern  Select the correct search pattern for the targeted satellite.
  • Inclined is for a satellite in an inclined orbit
  • Sky search is for no gyro mode
  • Sprial is for normal operation
 		 
Tx Polarity  Select the correct Transmite polarity
If in Xpol the the recieve will be the opersite pol 		 
Band  Select the correct band on the LNB for tracking and modem receive  
Reflector  Primary   Secondary This is for the Dual Band systems with C and Ku feeds or QOR antennas which have two reflectors. If the antenna is only C or Ku band only then secondary can not be selected.
Rx  XPol        CoPol This will switch between the two LNB's, Xpol will allow you to receive on the opersite pol to the TX and CoPol will allow you to receive on the same pol as the TX
