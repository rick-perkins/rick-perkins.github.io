This page is where you can download the full configuration of the system.
Click on the download button for the config INI file.
This page is where the user can upload the software.
  1. Select the .enc file by using the browse button.
  2. Click on the upload button.
  3. Progress information will appear and when the software is loaded the reboot system button will appear.
  4. Click on it and the system will reboot and use the software.
Please give it a try.
This button will allow the user to reboot the full system, once the system has rebooted the user will need to log back into the MXP.
This where the user can upload the SSL Certificate.
When installed it activates the padlock and the https protocol (over port 443) and allows secure connections from a web server to a browser.
This is where you can lock down the system by using a unique code.
More information will be comming soon on this function.
Enter in your companies details that way the vessel or visiting technicain has the installers or airtime providers details
None of the fields can be left blank and if incorrect information has been entered then a pop up window will apear.
This is where the passwords can be changed.
  • The dealer can change all the passwords but to change the dealer password the old one needs to be entered.
  • The SysAdmin can only change thier own password and the Users
  • The User can only change thier own password.
If the Dealer password is unknown then Cobham Satcom support will need to be contact for a temporary password, for this to be generated support will need the MAC address.
The MAC address can be viewed by logging in to port 2003.

It is recommended by Cobham Satcom that the passwords should always be changed from the default ones when the system is commissioned.
