System Configuration

All the settings on this page are related to the reflector and NOT the pedistal.
The Trims are linked to the reflector because you are trimming the pointing angle of the reflector to the satellite, not the pedistal.

The Reflector indicates which refletor is being used. To switch between the refletors use the Position Antenna page.

  • On a dual band system the primary reflector will be C-band and Secondary will be Ku-band.
  • On a single reflector system, the primary is Ku-band

When using the C-band linear/circualr feed the offset vaules are different to when using KU.
The C-band will be a 4 digit number which is found on the feed assembly, there is one for circular and one for linear.

Please hover over the different names for more info and change the settings.

Trim

Trims are calculated in the same way as the DAC. The only differance is you do not need to mutiply by 10 becasue we can use decimal places.
CL is greyed out which means its not editable.
The EL trim should be within +/- 3 degs, if not then there may be an issue with the ICU.
The homeflag setting is linked to the AZ trim, if the AZ trim is greater than 5 degs the system will adjust the home flag setting.

Auto Trim works in the same way as the DAC.
  • System must be tracking the satellite AGC abpve threshold.
  • If modem lock is enabled then the MXP must be getting modem lock from the Vsat modem.
  • Elevation must not be greater than 75 Degs
  • System NOT setup for Inclined Orbit.

Calculate will automaticly setup the threshold levels which can be viewed on the Position Antenna page, at the same time the system will retarget the satellite, this should be done once the system is trimed on satellite.

           

Tracking Sensitivity  

Tracking Sensitivity is linked to Dishscan and how many decisions per second is made, 50% is equal to one decision per two seconds.
Increasing this value will slow the dishscan tracking down.

  • When in auto, the pol will adjust when the GPS posistion changes or a new satellite is targeted.
  • When in manual pol the operator has full control but this could allow bad cross pol isolation to acoure.
