Satellite Search

This screen provides the primary means to target the desired saved satellite.
The Satellite Selection drop down list will display all the saved satellites that have been entered in and saved.Once the satellite has been selected, the antenna will re-target and track using the saved settings.
The Latitude, longtitude and Heading are read only.
To edit the saved satellites is done via the Configuration Satellite page.

Click on the drop down arrow to see the selections.
