Display Overview

The AGC numeric value is shown just above the signal bar.
The signal bar graph displays the calculated AGC threshold as an orange bar at the center.
The graph displays a range that extends from AGC Threshold -300 (at the left) to AGC Threshold +300 (at the right).
If the AGC is less than the minimum value all bars are grey.
AGC1570 + 300 = Threshold 1870
If the AGC is less than the threshold value bars are red
AGC1570 + 150 = Threshold 1620
If the AGC is greater than the threshold value bars are green
AGC1570 - 150 = Threshold 1420
If the AGC is greater than the maximum value all bars are green
AGC1570 - 300 = Threshold 1270
