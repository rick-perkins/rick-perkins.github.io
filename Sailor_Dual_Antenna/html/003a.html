Modem profiles

Add modem profile

Profile name
Modem
IP address . . .
  1. Log in to the slave ACU and go to Settings > VSAT modem profile > New entry.
    (With older software it will be Settings > Satellite Profiles > VSAT modem profile > New entry.)
  2. From the drop down list please select Dual Antenna Master.
  3. The profile name will update and can be changed if needed, for this training will will leave as Dual Antenna Master.
  4. Enter in the IP address of the master acu.
  5. Click Apply.
  6. Please click on Satellite profiles for the next step.
