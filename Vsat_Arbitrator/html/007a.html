The key settings are:
  • IP Address: This is the Arbitrator IP address, when using OpenAMIP the modem needs to send all the commands to this IP address.
  • Antenna A and B IP: These must be the same as the IP address of DAC A and B other wise the OpenAMIP commands will not get passed on to the DAC's.
  • Forward TCP Port: This must be set to the port which has been setup in the DAC's for OpenAMIP commands, default is 2002.
  • Mute output polarity: This must be set correctly depending on the VSAT modem I.E. Idirect will be high voltage mute and Comtech will be Low voltage mute.
To save the settings, Submit must be pressed before pressing save.
Please hover over the names of the fields for more information.
