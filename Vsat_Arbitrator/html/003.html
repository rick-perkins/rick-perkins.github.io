Connections to the modem

The RF is switched via relays. The common port is the one that always goes to the VSAT modem.
Transmit is labeled as TX and recieve is labeled as RX.
Click here to add the coax cables

Next is to connect the serial connection. This is for TX mute, RX lock and GPS. This serial connection is only needed when not using openAMIP or ROAM.
(Note: TX mute is not done by the DAC or MXP, it's controlled by the arbitrator only when using the serial connection.)
Click here to add the serial modem cable

Next is to connect the Ethernet cables.This is for the web interface, ROAM or openAMIP if used.
(Note: TX mute is passed from the active antenna to the modem via the arbitrator.)
Click here to add the Ethernet cables
