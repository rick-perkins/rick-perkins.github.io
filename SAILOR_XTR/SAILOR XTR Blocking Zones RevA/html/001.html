Overview

  • Blocking zones are used to allow the antenna to know if the satellite is lost due to a known issue I.E blockage rather then an issue with the antenna or satellite.
  • The blockage has to be a fixed structure to the vessel that will not move.
    • Funnel
    • Other antennas
    • Masts
    • Cranes
    • Areas that the system is not allowed to transmit though.
  • The system can be configured for up to 8 blocking areas.
  • Each area will have a start and stop azimuth range and elevation angle.
  • When the antenna is in this programed area the system can mute the modem if the zone is set for No TX.
  • Typically the blocking zone will be set for No TX (transmitting though blockage can cause transmission issues and reflections).
  • Switch to the second system if it’s a dual system.
  • The azimuth is calculated from 0° to 360° from the bow in a clockwise ↻ direction.
  • The elevation range is -30° to 90°.
  • Blocking zones can over lap each other from the largest to the smallest.
